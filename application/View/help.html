{% trans %}Q: What is MegaCrypter?

A: MegaCrypter is an open-source anonymous crypted url forwarding software for MEGA links. Its main goal is to protect your MEGA account encrypting your links BEFORE sending them through a channel you don't trust at all.{% endtrans %}

{% trans %}Q: How can i download MegaCrypter links?

A:

Option 1 (only domain-lock links): download from your browser (Java plugin is required and remember you must agree with TOS).

Option 2: use some download manager like Jdownloader 2 beta, Pyload, Mipony, DownDroid, PonyDroid (pay attention to the versions).

Option 3: use this bash script

Option 4: code your own download application using MegaCrypter public API.{% endtrans %}

{% trans %}Q: How safe is MegaCrypter?

A: Your MEGA links (and other sensitive data) travel encrypted over HTTPS On the other hand, MegaCrypter links use AES (256 bits) and are decrypted on the fly at server side. If you are concerned about your privacy, you may like to know that MegaCrypter DOES NOT LOG ANYTHING about you or your links.{% endtrans %}

{% trans %}Q: Can i protect MEGA folders?

A: Yes. MegaCrypter will return you a protected link for every file that is inside the folder at this moment.{% endtrans %}

{% trans %}Q: Why MC shows "Web access was not enabled for this link" ?

A: Because that link was created WITHOUT DOMAIN LOCK protection (see advanced options below) and it can be only downloaded using a download manager.{% endtrans %}

{% trans %}Q: Where are files downloaded from?

A: MEGA files are directly downloaded from MEGA cloud to your computer (MegaCrypter does not host nor stream any MEGA file).{% endtrans %}

{% trans %}Q: Ok, i have now Java working but it asks me to load Kissdownloader and it says it could be dangerous!

{% trans %}Q: I'm downloading a file with Kissdownloader... Can i stop and resume download later?

A: yes, you can (even closing your browser), but before quitting you MUST push STOP DOWNLOAD button. (Just remember next time, you must use the same filename if you want to resume your previous download).{% endtrans %}

{% trans %}Q: What are slots?

A: Download slots are very useful if you want to maximize your download speed. Try increasing them step by step until your download speed is maximum, but keep in mind that slots consume CPU/memory resources and might produce network overhead, so try to keep slots value as minimum as possible.{% endtrans %}

{% trans %}Q: Can i download several files at the same time?

A: Of course you can ;){% endtrans %}

{% trans %}Q: How does file integrity check work?

A: It's quite simple. AFTER download is finished, Kissdownloader generates file's CBC-MAC and compares it against original files's CBC-MAC (which is derivated from mega link key). If both codes are equal, your file is (99.9% prob) bit-by-bit identical to the original.{% endtrans %}

{% trans %}Q: Kissdownloader says my file is damaged. What does it mean?

A: There are basically two scenarios for this error:

Scenario 1: Your downloaded file was corrupted due some Internet connection problem (try to download again but if error persists, you are probably in scenario 2).

Scenario 2: MEGA file was corrupted during upload (file must be re-uploaded).{% endtrans %}

{% trans %}Q: What does it mean file temporarily unavailable?

A: It means what it says xD Jokes apart, this "random" error it's related to MEGA servers maintenance. It can be fixed in a couple of minutes, hours or days...{% endtrans %}

{% trans %}Q: Kissdownloader loads but download speed keeps frozen or very slow. Why?

A: Some MEGA files are stored in more overloaded servers. This problem is related with "file temporarily unavailable" and it appears and fixes itself apparently in random. If download speed is frozen you can wait for it to start or cancel download (be patient while download is stopping), but if download speed is slow, you can try to increment slots.{% endtrans %}

{% trans %}Q: What about video streaming?

A: You can watch protected video files directly from web with VLC and Java. Before pressing PLAY button in VLC player, you MUST wait for Kissvideostreamer to load (It's necessary to decrypt MEGA video files on the fly). And, keep in mind that it's still experimental.{% endtrans %}

{% trans %}Q: If i don't want some MegaCrypter link to be active anymore, can i disable it?

A: MegaCrypter links can't be directly disabled but you can delete or move to another folder your original MEGA file associated with some MegaCrypter link to disable it (and restore it to enable MegaCrypter link again). Remember you can also set an expiration time value when you create MegaCrypter links (see advanced options).{% endtrans %}

{% trans %}Q: Could you explain MegaCrypter links advanced options?

A:

Expire: By default MegaCrypter links are eternal (unless original MEGA file is removed), but, in case you need it, you can set an expiration time value from 10 minutes to one year (if your MegaCrypter link expires, your original MEGA file WILL NOT be affected in any way).

Append file info: this option is useful in case you protect a lot of links (or an entire folder) because you won't have to add manually file names to every protected link. (Off by default)

Return short urls: If you need crypted short urls use this option (Off by default).

Mask filenames even at download time: MegaCrypter will hide file name except file extension or episode numbers. (It can be useful against web crawlers).

Domain lock: Using this option, your MegaCrypter links only will be accessible by cliking from some particular domain name (they still can be downloaded using a download manager).

Extra info: Put here any interesting information about your links (64 characters max).

Email: Put here an email to alert you in case that something happens with your links (64 characters max).

Password: You can add an extra security layer to your MegaCryper links. (Just remember passwords are case sensitive).{% endtrans %}
